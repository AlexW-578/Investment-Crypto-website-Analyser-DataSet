    Microsoft SwiftKey für Android

    Microsoft SwiftKey ist lernfähig und erkennt Ihren individuellen Schreibstil, unter anderem auch die Wörter, Wendungen und Emojis, die Sie häufig verwenden. Dadurch erhalten Sie genauere Autokorrekturen und Wortvorschläge.
    Microsoft SwiftKey-Video abspielen

    Ihr persönlicher Stil

    Wählen Sie aus Hunderten kostenloser Tastaturdesigns, oder entwerfen Sie Ihr eigenes Design.

    Tippen ohne Frustration

    Microsoft SwiftKey beschleunigt und vereinfacht das Tippen mit einer Vielzahl von Funktionen.

    Wischen statt Tippen

    Tippen ist zu umständlich? Mit Microsoft SwiftKey Flow können Sie einfach von Buchstabe zu Buchstabe wischen.

    Schnell und präzise

    Schluss mit Tippfehlern. Microsoft SwiftKey korrigiert Tippfehler, falsche Leerzeichen und fehlende Buchstaben für Sie.

    Personalisierte Eingabe

    Passen Sie die Symbolleiste mit Ihren bevorzugten Eingabe-Tools an. Darunter GIFs, Zwischenablage, Übersetzer, Sticker und mehr.

    Lernt von Ihnen

    Microsoft SwiftKey lernt Ihren Schreibstil und schlägt das passende nächste Wort vor. Dadurch können Sie ein ganzes Wort einfach per Tippen eingeben anstatt jeden einzelnen Buchstaben.

    Schreiben Sie in fünf Sprachen

    Sie können gleichzeitig bis zu fünf Sprachen verwenden, ohne Einstellungen ändern zu müssen. Wählen Sie aus über 400 unterstützten Sprachen.
