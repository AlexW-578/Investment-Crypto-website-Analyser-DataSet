Current Conditions

  • The AccuWeather API provides subscribers access to location based weather data via a simple RESTful web interface.
  • Subscribers may search for current condition observations for locations world wide.
  • Current condition searches require a location key. Please use the Location API in order to obtain the location key for your desired location.
  • Historical current condition data (observations from the past 6 or 24 hours for a particular location) is also available.
  • Data is available in more than 200 languages and dialects.
  • By default, a truncated version of the current condition data is returned which includes: observation datetime, weather phrase, weather icon, precipitation indicator flags, and temperature. The full object can be obtained by passing "details=true" into the url string.
  • Current conditions may be obtained for the world's top cities in lists of 50, 100, or 150 cities.
